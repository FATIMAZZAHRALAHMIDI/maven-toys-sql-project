
# maven-toys-sql-project
# Toys Sales Analysis using SQL

## Project Overview
This project analyzes toy store sales data using MySQL.

## Dataset
- Sales
- Products
- Stores
- Inventory

## Skills Demonstrated
- Data Cleaning
- Data Modeling
- CTEs
- Window Functions
- Business Analysis
- Revenue Analysis

## Key Business Questions
1. What is the total revenue per year?
2. Which products generate 80% of revenue?
3. Which stores perform best?
4. Which months generate the highest sales?

## SQL Concepts Used
- JOIN
- CTE
- CASE WHEN
- LAG
- RANK
